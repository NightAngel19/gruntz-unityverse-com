Exit Trigger
The ExitTrigger game logic is always assigned to the LEVEL_FORT and is used as the exit in a Questz game or as a player's fort in a Battlez game.
Edit Objects Dialog Box
Logic: ExitTrigger
Image Set: LEVEL_FORT
Smarts: 0 - Player 1 fort (only required for Battlez)
1 - Player 2 fort (Battlez only)
2 - Player 3 fort (Battlez only)
3 - Player 4 fort (Battlez only)
