BehindCandy and BehindCandyAni
This is one of the few Gruntz logicz that take advantage of the Object Flags Dialog Box.

Because of this, I will list the purpose of this entry first.
Draw Flags (useful for Gruntz)
No Draw Checked
Mirror Checked
Unchecked		 image will be 'flipped' horizontally.
image will be 'normal'.
Invert Checked
Unchecked		 image will be 'flipped' vertically ... but look very strange.
image will be 'normal'.
Flash Checked
Unchecked		 image will flash (discouraged ... very annoying).
image will be 'normal'.


BehindCandy
Edit Objects Dialog Box
Logic: BehindCandy
Image Set: Example: LEVEL_ROCKZ1
Animation: Empty, used only by BehindCandyAni and EyeCandyAni type Objectz
This should be set to any image set name that is intended to be an eye-candy that you want to display behind all Gruntz.
BehindCandyAni
Edit Objects Dialog Box
(same as BehindCandy, except for the entriez below)
Logic: BehindCandyAni
Image Set: Example: LEVEL_WATER1
Animation: Example: GAME_WATER1
This should be set to any animated image set name that is intended to be an eye-candy that you want to display behind all Gruntz.
EyeCandy
Edit Objects Dialog Box
(same as BehindCandy, except for the entriez below)
Logic: EyeCandy
Image Set: Example: LEVEL_TREE1
This should be set to any image set name that is intended to be an eye-candy that you want Gruntz to be able to walk behind.
EyeCandyAni
Edit Objects Dialog Box
(same as BehindCandy, except for the entriez below)
Logic: EyeCandyAni
Image Set: Example: LEVEL_TREE2
This should be set to any image set name that is intended to be an animated eye-candy that you want Gruntz to be able to walk behind. I have not experimented enough with EyeCandyAni to know if there are any EyeCandy objectz that are animated, or not.

Image set names used in Gruntz
Imagez designed to be BehindCandy(Ani), or EyeCandy(Ani) which exist in every graphicz set.
Imagez designed to be BehindCandy(Ani), or EyeCandy(Ani).
Other imagez that may be used as BehindCandy(Ani), or EyeCandy(Ani).
Use these at your own peril.
