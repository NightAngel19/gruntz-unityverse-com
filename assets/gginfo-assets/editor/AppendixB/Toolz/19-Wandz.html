19 - Magic Wandz

Method number 1: Enter the game with it.
Edit Objects Dialog Box
Use Logic: GruntStartingPoint with Powerup: = 19 ... other fieldz are necessary!
There is a specialized enemy Grunt called a Magic Wand Grunt that may be defined simply by providing a "Points: value of 15 and "Smarts: value of 1, 2 or 3 in this method.
Supposedly, "Face Dir:" allowz Spell selection, but only "Random Spellz" result (and usually only 1, 2 or 3 are read).
Try placing an InGameIcon 'owned' by the enemy Grunt on his GruntStartingPoint, to set a specific spell.

Method number 2: Cover it up.
Edit Objects Dialog Box
Use Logic: CoveredPowerUp with Powerup: = 19 ... other fieldz are necessary!

Method number 3: Leave it in plain sight.
Edit Objects Dialog Box
Use Logic: InGameIcon with Image_Set: = GAME_INGAMEICONZ_TOOLZ_WANDZ
... other fieldz are necessary, especially "Face Dir:"! (Workz to select the spell cast.)

Method number 4: Put it in a Giant Rock.
No matter what 'team' they are on, a Gauntletz Grunt, (Suicide) Bomber Grunt or Time Bombz Grunt can break or blow up a Giant Rock, or a Shovel Grunt can dig up a Time Bomb next to a Giant Rock, revealing the Tool in the middle, where Tile #290 was placed.

Method number 5: Ask for it.
One of the solver's Gruntz can use a Megaphone to call for the Tool.

Method number 6: Steal it (back).
Turn a Tool Thief carrying the Tool into a Goo Puddle , then pick up the Tool he leavez behind.
Using this Tool sapz the Grunt's health, losing 25% of original full health each time a Spell is cast.
Do not give this Tool to a Resurrected Grunt, unless there is also at least a Can of Zap Cola handy.
(A Bottle or Keg would be even better!)
Travel speed: 600 millisecondz per Tile ... 300 millisecondz per Tile with the Super Speed PowerUp.
There are six possible Spellz generated by a Random Spell Magic Wand (see the list).
