Alternate 'Ground' Tilez
Tile Properties/Attributes: Clear

Only Tilez 90 and 91 exist for all graphics sets.
The Areas in which the other Tilez exist are listed with the Tile descriptions.
Graphics Area number
Tile 1 2 3 4 5 6 7 8
90 light-colored background Tilez 1-8.
91 dark-colored background Tilez 9-16.
92 landing area Tile (Area 5, 6, north) _ (Area 8)
93 landing area Tile (Area 5, 6, west) _ (Area 8)
94 landing area Tile (Area 5, 6, east) _ (Area 8)
95 landing area Tile (Area 5, 6, south) _ (Area 8)
These Tilez are usually treated as being on the same elevation, requiring no stairways to go from any one of them, to any other one of them.

Note: Specifying a Tile from this set that showz an error image within a CoveredPowerup or GiantRock logic will result in Gruntz loading failure.

It is a FATAL error to use an invalid Tile number ... anywhere!
